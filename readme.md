# 简易的计算器

> 在线使用

[简易的计算器](http://clovey.party/calculator/build/index.html)

## 介绍

一个建议的计算器。

## 特点

1. 实现基本的加减乘除、取余功能。
2. 内置部分数学函数，如sin(),cos(),tan(),abs(),pow(),ln()等
3. 内置部分常量，如pi、e等
4. 能够处理圆括号运算，如：“（1+2+3-1）”。
5. 能够处理重复加减符号，如：“-1++++2”。
6. 在表达式出错时，能够大致指出错误位置！
